## 🚀 الخطوات التالية

### الخطوة 1: تشغيل السيرفر
```bash
php artisan serve
```
ستفتح الموقع على: **http://localhost:8000**

### الخطوة 2: إنشاء عنبر أول
1. اضغط على "إضافة عنبر جديد"
2. أدخل اسم العنبر (مثال: عنبر 1)
3. اضغط "حفظ"

### الخطوة 3: إنشاء دورة جديدة
1. اضغط على تفاصيل العنبر
2. اضغط "إنشاء دورة جديدة"
3. أدخل:
   - تاريخ البداية
   - عدد الكتاكيت الأولي (مثال: 1000)
4. اضغط "حفظ"

### الخطوة 4: تسجيل النافق
1. في صفحة الدورة اضغط "إضافة نافق"
2. أدخل عدد النافق المراد إضافته
3. اضغط "حفظ"
* النظام يحسب المتبقي تلقائياً ✨

### الخطوة 5: إضافة سجلات مالية
1. في صفحة الدورة اضغط "إضافة سجل مالي"
2. اختر النوع (مصروف أو إيراد)
3. أدخل المبلغ والوصف والتاريخ
4. اضغط "حفظ"
* النظام يحسب الإجمالي تلقائياً ✨

### الخطوة 6: إغلاق الدورة
1. في صفحة الدورة اضغط "إغلاق الدورة"
2. أدخل عدد الكتاكيت المباعة الفعلي
3. النظام سيكشف الفروقات تلقائياً ✨
4. اضغط "إغلاق الدورة"

### الخطوة 7: عرض التقارير السنوية
1. اذهب إلى القائمة → السجلات المالية
2. اختر السنة
3. اضغط "تحديث"
4. يمكنك طباعة التقرير مباشرة ✨

## 📊 المميزات المدمجة

### الحسابات التلقائية 🧮
- **الكتاكيت المتبقي** = الأولي - النافق
- **المصروفات الإجمالية** = مجموع المصروفات
- **الإيرادات الإجمالية** = مجموع الإيرادات
- **صافي الحاصل** = الإيرادات - المصروفات
- **هامش الربح** = (صافي الحاصل ÷ الإيرادات) × 100

### كشف الفروقات 🔍
عند إغلاق دورة:
- إذا تطابق المبيعات مع المتبقي: ✅ "بدون فروقات"
- إذا كانت المبيعات أقل: ⚠️ "خسائر غير مسجلة"
- إذا كانت المبيعات أكثر: ⚠️ "مبيعات أكثر من المتوقع"

## 🎯 حالات الاستخدام

### الحالة 1: تتبع دورة واحدة
```
عنبر → دورة جديدة → إضافة نافق يومي → إضافة مصروفات يومية → إغلاق الدورة
```

### الحالة 2: مقارنة أداء عدة دورات
```
التقرير السنوي → مقارنة المربحية بين الدورات
```

### الحالة 3: تحليل المشاكل
```
عند اكتشاف فروقات عند إغلاق الدورة → النظام يخبرك بالخسائر الإضافية
```

## 💡 نصائح
1. **أدخل البيانات يومياً**: للحصول على دقة أفضل
2. **تحقق من الفروقات**: عند الإغلاق قد تكتشف مشاكل
3. **احفظ التقارير**: استخدم طباعة التقرير السنوي للأرشيفة
4. **راجع الهامش**: هامش الربح العالي يعني كفاءة جيدة
